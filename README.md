## 特色

1. 快速計算 GPA
2. 支援暗黑模式
3. 支援 RWD
4. 支援 GPA 4.3 與百分制（依臺科規則）

## 如何使用

### [→ 網址點我 ←](https://gpa-calculator.vicwen.app/)

根據積分個數與成績來快速計算 GPA。

## 本機開發

```bash
pnpm install
pnpm dev
```

## 建置

```bash
pnpm build
```

## 相關事項

1. 本專案僅前端，不用擔心你的超高 GPA 會被我看到。
2. 本專案是以 GPA 4.3 標準為計算基礎。
3. 有任何 bug 或建議，歡迎開 Issue 或 PR。
